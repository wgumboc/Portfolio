About Me

Welcome! My name is Will and I am currently a Computer Science student at The University of British Columbia in the BCS program. Previous to this, I graduated with a degree in Chemical Engineering and worked as a quality control supervisor for a beverage company. During this time, I would noticed many unoptimized processes and would create VBA or Python scripts to automate repetitive tasks, sparking my interest in programming. I started tinkering with JavaScript, HTML, and CSS, and was instantly hooked. After completing many courses throughout my CS journey, learning knew skills, and working on more complex projects, the fire I had for programming burns even brighter. I find joy in continuous learning and am always looking for ways to develop myself.

In my free time I enjoy staying active in the outdoors through soccer, skiing, and hiking. I also enjoy annoying my friends and family by singing and freestyling on the piano.

Projects


Course Progress Dashboard

I created this web app using a React frontend with an Express backend framework that allows a user to track their courses and assessments, calculate their weighted average, and predict their final grade through the use of function components. I set up and connected a MongoDB database to enable data to be stored and retrieved for user specific course addition and removal, custom assessment addition, and continuous grade calculation. User authorization was implemented with JSON Web Token and authentication was achieved using Redux to manage state. Data was updated using Axios to make CRUD requests to a REST API.

Parkpal

This team project was completed for BCS Hacks 2022. We created a web app that visually locates the 100 closest parking meters to any selected spot in Vancouver through querying data provided by the City of Vancouver's parking API as well as utilization of the Google Maps API. The app also features the ability to highlight the 10 cheapest spots as well as filter for spots that allow stays of > 2hrs.

Manning Scheduler

As a previous supervisor who did scheduling using excel, I decided to optimize the process by creating a Java desktop application using the Swing GUI toolkit for creating new positions and assigning employees to their relevant position restricted by their skillset.

Tic-Tac-Toe
