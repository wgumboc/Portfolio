Course Progress Dashboard

I created this web app using a React frontend with an Express backend framework that allows a user to track their courses and assessments, calculate their weighted average, and predict their final grade through the use of function components I set up and connected a MongoDB database to enable data to be stored and retrieved for user specific course addition and removal, custom assessment addition, and continuous grade calculation. JSON Web Token was used for user authorization and authentication was implemented using the Redux toolkit. To update data, I used a REST API with CRUD operations.
